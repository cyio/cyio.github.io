# 常见错误

  1. param is empty:jsApiList 中没有参数 注意：你需要使用的所有的API都需要已数组元素的形式写入该参数中

  2. invalid url domain：无效的域名 注意：appid和appsecret是否写正确，JS接口安全域名是否正确

  3. require subscribe 错误说明你没有订阅该测试号，该错误仅测试号会出现

  4. 桌面端先调试，后移动端
