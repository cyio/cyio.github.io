# Web 原生组件

适用场景

自定义元素的主要好处是，它们可以在使用任何框架，甚至是在不使用框架的场景下使用。当你面向的最终用户可能使用了不同的前端技术栈，或是当你希望将最终的应用与它使用的组件实现细节解耦时，它们会是理想的选择。

# Web Components vs. Vue 组件

优点：使应用“永不过时”

弊端：模板系统弱些，状态管理系统，SSR 效率（模拟 dom）不如框架（输出 string）
