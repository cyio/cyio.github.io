# Canvas

Canvas 是基于状态的绘制 将状态设置与确定绘制的代码分隔开

# 基础

stroke 描边

# arc 弧形

const canvas = document.getElementById('canvas')
const ctx = canvas.getContext('2d')

ctx.beginPath()
ctx.fillStyle = '#000'
ctx.arc(75, 75, 50, 0, 2 * Math.PI) // 优化，把值单独提前声明
ctx.fill() // 实心，空心用 stroke
ctx.closePath()
