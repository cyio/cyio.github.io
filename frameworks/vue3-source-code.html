# reactive

文件结构

└── src
    ├── baseHandlers.ts // 基本类型的处理器
    ├── collectionHandlers.ts  // Set Map WeakSet WeckMap的处理器
    ├── computed.ts // 计算属性，同Vue2
    ├── deferredComputed.ts // 计算属性
    ├── dep.ts // 
    ├── effect.ts // reactive 核心，处理依赖收集，依赖更新
    ├── effectScope.ts // 
    ├── index.ts
    ├── operations.ts // 定义依赖收集，依赖更新的类型
