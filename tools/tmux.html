# 滚屏模式

默认状态下不能滚屏，进入：前缀键 + [ 滚动：vi 按键风络或上下方向键或 PgUp/PgDown 复制：prefix -> [ -> v -> y -> q 粘贴：prefix -> ]

# 会话

# 创建会话
tmux new -s [NAME]
# 挂载会话
tmux attach
# 分离会话
tmux detach
# 列出会话
tmux ls
# 杀掉会话
tmux kill-session -t [ID]
# 保存会话
`prefix ctrl s`
# 恢复会话
`prefix ctrl r`
