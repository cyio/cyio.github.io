# VuePress

> VuePress. Vue-powered Static Site Generator ... VuePress generates pre-rendered static HTML for each page, and runs as an SPA once a page is loaded.

基于 Vue 的静态网站生成器（SSG），为每个页面生成预渲染 HTML，页面加载后以 SPA 运行。

- 某个页面不显示侧边栏

```
---
sidebar: false
---
```
