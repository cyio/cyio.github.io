# 应用内购买

StoreKit 应用中嵌入商店

用户，应用，苹果商店三方之间

## 流程

1. 取回产品信息

- 获取产品标识符列表，可打包进应用，可从服务器获取，最好做版本管理
- 取回产品信息
- 展示

2. 发起支付

3. 等待并标记支付完成
