>  软件结构是指软件系统的基本结构，以及创建这种结构和系统的约束。每个结构包括软件元素、元素之间的关系，以及元素和关系的属性。软件系统架构是一个比喻，类似于建筑物的架构。它作为系统和开发项目的蓝图，列出了设计团队需要执行的任务。 - 维基百科

> 架构是指体现在它的组件中的一个系统的基本组织、组织之间的关系、组织与环境的关系及指导其设计和发展的原则。 - IEEE
