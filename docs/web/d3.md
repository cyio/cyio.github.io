* 数据 => DOM，HTML/CSS/SVG
* 解决关键问题：基于数据高效操作 DOM

## 与其它库对比
Highcharts、ECharts层级比D3靠上，D3更接近绘图引擎方向。通常是前两个搞不定的图，就让第三个来。D3比前两个自由度更高些，所以需要掌握的知识也更多些（比如计算机图形学什么的），使用前两个的时候更多关心的是业务。
